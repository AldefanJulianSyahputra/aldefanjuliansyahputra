Tentang Web Ini

Website ini dibuat sebagai media pembelajaran dalam memahami dasar-dasar HTML, CSS, dan penggunaan hyperlink, serta untuk latihan proses hosting web sederhana.

Tujuan Pembuatan

  • Belajar membuat halaman web sederhana menggunakan HTML.
  • Mengenal penggunaan hyperlink antar halaman.
  • Mempelajari dasar styling dengan CSS dan Bootstrap.
  • Melatih kemampuan mengupload website ke hosting.

Teknologi yang Digunakan

HTML, CSS,JavaScript dan Bootstrap 5
